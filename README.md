# 🍁 Vancouver Neighbourhood Safety
An interactive Streamlit dashboard for exploring Vancouver crime data by neighbourhood, crime type, and time of day.


## Requirements

- Python 3.11
- pip

---

## Installation

**1. Clone the repository**

```bash
git clone https://github.com/Brymahh/streamlit_deployment_vancouver_neighbourhood_safety.git
cd streamlit_deployment_vancouver_neighbourhood_safety
```

**2. Install the project environment

```bash
conda env create -f environment.yml
conda activate streamlit_env
```

**3 Running the Application**

```bash
streamlit run app.py
```

The app will open automatically in your browser at `http://localhost:8501`
